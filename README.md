# Sentiment Analysis on Movie Reviews 

This project uses IMDB's movie review dataset from Kaggle. Naive
Bayes classifier was used to classify the reviews as positive or negative with 85% accuracy. Used
NLP techniques like tokenization, lemmatization, tf-idf vectorization for data pre-processing.

- Python
- Machine Learning
- Naive Bayes Classfier
- Natural Language Processing
